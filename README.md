# D&D Campaign Manager

A web-based tool for managing D&D campaigns, sessions, and character tracking.

## Features

- Campaign & World Building
- Session Management
- NPC Tracker
- Player & Character Tracking
- Combat & Encounter Tools
- Loot & Inventory
- Rules Reference
- Notes & Secrets
- Timeline / History Log
- Random Generators

## Getting Started

This project is built with React and uses npm for package management.

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Building

```bash
npm run build
```

## Project Structure

- `src/` - Source code
- `public/` - Static assets
