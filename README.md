# D&D DM Tool

A web-based tool for Dungeons & Dragons Dungeon Masters to track players, monsters, encounters, and campaign notes.

## Features

- **Party Management**: Track character sheets, equipment, and status
- **Monster Tracker**: Monitor initiative, health, and spell slots
- **Encounter Planner**: Plan and balance encounters with random elements
- **Campaign Notes**: Organize adventure notes and location descriptions
- **Dice Roller**: Virtual dice with animations and results
- **Quick Stats**: Generate random numbers, critical hits, save throws

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Tech Stack

- React 18
- Vite 6
- React Router
- Shadcn/ui
