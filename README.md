# D&D Character Manager

A comprehensive companion tool for Dungeons & Dragons players and Dungeon Masters.

## Features

- Campaign & World Building
- Session Management
- NPC Tracker
- Player & Character Tracking
- Combat & Encounter Tools
- Loot & Inventory Management

## Tech Stack

- **Frontend Framework**: React 18.3.1
- **Styling**: CSS (custom)
- **Build Tool**: Create React App (react-scripts)
- **State Management**: Local component state (planned: Context API or Redux for complex state)

## Getting Started

### Prerequisites

- Node.js (version 16 or higher recommended)
- npm (version 9 or higher)

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

```bash
npm install
```

### Development

To start the development server:

```bash
npm start
```

This will start the app in development mode.

### Building

To build the app for production:

```bash
npm run build
```

### Testing

To run tests:

```bash
npm test
```
